# PharmaX - نظام إدارة المخازن والمبيعات

## نظرة عامة

PharmaX هو برنامج محاسبة متكامل لإدارة المخازن والمبيعات مصمم خصيصاً للسوق السعودي. يوفر النظام حلولاً شاملة لإدارة العملاء، المخزون، المبيعات، وإصدار الوصولات مع إمكانية التخزين المحلي والمزامنة السحابية المستقبلية.

## المميزات الرئيسية

### 🏪 إدارة المخزون

- إضافة وتعديل وحذف المنتجات
- تتبع كميات المخزون
- تنبيهات للمنتجات منخفضة المخزون
- إدارة الفئات والباركود
- حساب التكلفة والسعر

### 👥 إدارة العملاء

- سجلات العملاء الشاملة
- معلومات الاتصال (هاتف، بريد إلكتروني)
- العناوان ومواقع المشاريع
- حسابات العملاء والرصيد الحالي
- حدود الائتمان

### 💰 نظام المبيعات

- إنشاء فواتير البيع بسهولة
- دعم طرق الدفع المختلفة (نقدي، آجل، بطاقة)
- حساب الخصومات والضرائب تلقائياً
- تتبع المدفوعات المعلقة
- تقارير المبيعات الشاملة

### 🧾 إصدار الوصولات

- قوالب وصولات جاهزة
- إصدار PDF فوري
- تخصيص قوالب الوصولات
- أرشفة الوصولات الرقمية

### 💾 التخزين والمزامنة

- تخزين البيانات محلياً (SQLite)
- هيكل جاهز للمزامنة مع Firebase
- نسخ احتياطي آمن
- استعادة البيانات

## المتطلبات التقنية

- Java 17 أو أعلى
- Maven 3.6+
- JavaFX 17
- SQLite
- (مستقبلاً) Firebase

## التثبيت والتشغيل

### 1. استنساخ المشروع

```bash
git clone <repository-url>
cd windsurf-project
```

### 2. بناء المشروع

```bash
mvn clean compile
```

### 3. تشغيل التطبيق

```bash
mvn javafx:run
```

أو بناء ملف Jar قابل للتنفيذ:

```bash
mvn clean package
java -jar target/inventory-management-1.2.4.jar
```

## قاعدة البيانات

يستخدم النظام SQLite للتخزين المحلي مع الهيكل التالي:

- **customers**: معلومات العملاء
- **products**: بيانات المنتجات والمخزون
- **sales**: سجلات المبيعات
- **sale_items**: تفاصيل بنود البيع
- **receipts**: سجلات الوصولات الصادرة

## التكامل المستقبلي مع Firebase

النظام مهيأ بالكامل للتكامل مع Firebase لتوفير:

- المزامنة السحابية للبيانات
- النسخ الاحتياطي التلقائي
- الوصول متعدد المستخدمين
- التقارير السحابية

## دعم اللغة العربية

النظام مصمم بالكامل لدعم اللغة العربية:

- تقارير ووصولات باللغة العربية
- دعم النصوص العربية في قاعدة البيانات
